Реализовать UART bridge в рамках одного STM32 микроконтроллера под FreeRTOS. 
Данные принимаемые на RX первого UART должны передаваться на TX второго UART и наоборот.
Ограничения. Работа с UART должна выполняться по прерываниям, как для приема так и для передачи. 
Дополнительно, обработка данных каждого из UART должна происходить в своей отдельной FreeRTOS задаче.
Результат - в виде Github репозитория.
